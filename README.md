Python Turtle Hangman
Welcome to our Hangman game! We built this fun guessing game using Python.
How It Works
In this game, you try to guess a secret word by picking one letter at a time. If you guess wrong, a piece of a stick figure is drawn. If the whole stick figure is drawn before you guess the word, you lose!
Fun Things We Added
Welcome Screen: Shows the rules for a few seconds before the game starts.
Draws the Hangman: Watch the stick figure get drawn step-by-step when you make a bad guess.
Smart Typing Check: It will give you a warning if you type numbers, double letters, or a letter you already picked.
Win and Lose Screens: Enjoy a colorful screen if you win, or a funny screen if you lose.
Play Again Fast: Just click anywhere on the screen when the game is over to start a new game.
How to Play
Run the hangman (8).py file on your computer.
Read the welcome screen.
When the box pops up, type one letter and press enter.
Keep guessing to fill in the blank lines.
You have 6 chances to guess wrong before the game ends.
The Word List
The game will pick a random secret word from this group:
apple, orange, pear, pineapple, bear, cranberry, sunshine, time, enigma.
